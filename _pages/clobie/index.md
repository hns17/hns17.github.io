## 정리 방식

### 입력
- 디스코드 창작방의 원본 메시지
- 원본 채널, 날짜, 메시지 링크, 첨부 이미지

### 재구성
- **유형**: 설정 / 스토리 / 메모 / 캐릭터 / 배경 / 콘셉트 등
- **장르/분위기**: 판타지, SF, 감성, 미스터리, 몽환, 다크 등
- **시리즈**: 같은 세계관이나 연작을 묶는 단위
- **태그**: 키워드 기반 탐색

## 페이지 구성 원칙

1. **보관과 감상을 분리**합니다.
   - 보관: 원본 날짜, 원본 채널, 메시지 링크 유지
   - 감상: 제목, 소개문, 장르/유형, 시리즈 연결 강화
2. **글과 그림은 같은 UI로 억지로 맞추지 않습니다.**
   - 글: 카드형 아카이브 + 유형 분류
   - 그림: 갤러리형 썸네일 + 개별 작품 상세
3. **유형을 기본 축으로, 장르/분위기를 보조 축으로 사용**합니다.

## 현재 상태

아직 실제 항목은 비어 있지만, 구조는 아래 순서로 확장할 예정입니다.

- 글 컬렉션 구축
- 그림 컬렉션 구축
- 유형별 진입 페이지 구성
- 샘플 작업 몇 개 이관 후 표현 방식 조정
