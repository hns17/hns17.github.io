# Repository Instructions

## Post Authoring

- When creating or editing posts under `_posts`, read `POST_STYLE_GUIDE.md` first.
- Treat `POST_STYLE_GUIDE.md` as the source of truth for post structure and rewriting.
- Follow the guide for heading hierarchy, summary blocks, comparison tables, endings, and general readability cleanup.
- Do not add a duplicate body `#` heading when the page title already exists in front matter.
- For comparison posts, prefer quick-scan tables and use light emoji signaling only when it improves readability.
